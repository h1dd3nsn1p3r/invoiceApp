## Setup

1. Clone the repo.
2. Run `pnpm install` to install all the dependencies.
3. Run `pnpm run dev` to start the development server.

### Usage (API)

To generate invoice:

```sh
POST https://invoice.creamcode.com.np/api/v1/invoice HTTP/1.1
```
